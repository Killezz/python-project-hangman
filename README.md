# Tech/framework used

# Installation and running

```
# clone repo
git clone https://github.com/Killezz/python-project-hangman.git
cd python-project-hangman

# install requirements
pip install -r requirements.txt

# start backend
python backend/app.py

# start frontend
python hangman.py


```
